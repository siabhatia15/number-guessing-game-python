# 🎮 Number Guessing Game in Python

## About
This is a simple command-line Number Guessing Game built using Python. The program generates a random number, and the user has to guess it. The game provides hints until the correct number is guessed.

## Features
- Random number generation
- User input
- "Too High" and "Too Low" hints
- Beginner-friendly project

## Technologies Used
- Python 3

## Concepts Used
- Variables
- Loops
- Conditional Statements
- User Input
- `random` module

## How to Run

```bash
python guessing_game.py
```

This project helped me improve my understanding of Python basics, loops, conditionals, and problem-solving.
